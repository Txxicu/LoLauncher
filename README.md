# LoLauncher Version — Современный Minecraft Launcher

![Kotlin](https://img.shields.io/badge/Kotlin-1.9.22-purple)
![Version](https://img.shields.io/badge/Version-1.0.1-yellow)

**LoLauncher Version** — современный лаунчер Minecraft для ПК с жёлтой фирменной темой.

## Возможности

| Функция | Описание |
|---------|----------|
| Запуск игры | Release, Snapshot, Beta, Alpha |
| Модлоадеры | Forge, Fabric, **OptiFine** |
| Авторизация | Офлайн + Microsoft OAuth |
| Скины | Автозагрузка при вводе ника |
| Установка версий | Кнопка «Установить» → «Играть» |
| Моды | Просмотр, удаление, папка mods |
| Java | Автообнаружение и установка |
| Темы | Тёмная / светлая с перезапуском |
| Анимации | Можно отключить в настройках |
| Логи | Ошибки в `%APPDATA%/LoLauncher/logs/` |

## Запуск

```powershell
.\gradlew.bat run
```

## Создатель

**Txxicu Team**

## Логи

- `launcher.log` — ошибки лаунчера
- `game-output.log` — вывод Minecraft

Путь: `%APPDATA%/LoLauncher/logs/`
